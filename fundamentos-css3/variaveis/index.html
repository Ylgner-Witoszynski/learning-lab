Variáveis no CSS

Como declarar e usar variáveis no CSS

As variáveis do CSS são usadas para armazenar valores (como cores, tamanhos e espaçamentos) que podem ser reutilizados em diferentes partes do código. Dessa forma, é possível mudar toda a aparência do site alterando apenas um valor.
