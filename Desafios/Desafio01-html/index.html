Demonstração de Tags

Formatação de Texto Básica

Neste parágrafo, vamos utilizar várias tags de formatação. Podemos deixar o texto importante (strong), em itálico (i), ou sublinhado (u).

Também é possível realçar uma parte do texto (mark) para chamar atenção, ou inserir um texto de rodapé com a tag small (pequeno).

Novas Tags Pesquisadas

Abaixo estão as aplicações das tags sugeridas para pesquisa:

  • Abbr: Usada para abreviações. Passe o mouse sobre a sigla: Estou estudando HTML.
  • Del: Usada para indicar texto removido ou deletado. Exemplo: Preço: R$ 50,00 por R$ 25,00!
  • Font: Usada para cor e fonte (Obs: Esta tag foi descontinuada no HTML5, recomenda-se usar CSS, mas aqui está o exemplo solicitado):
    Este texto está vermelho usando a tag font.
